#define kGDMaxDisplays   12

Maximum online or active displays.

GDQuartzDisplay uses the core graphics API to get online/active displays by calling CGGetActiveDisplayList() and CGGetOnlineDisplayList(), this definition is the count that will be used when calling those functions.
