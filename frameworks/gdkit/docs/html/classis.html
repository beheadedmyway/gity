Detailed Description

The GDApplicationInfo which can load the main bundle's info plist, or any plist you specify.

You can use this to extend, or write extensions to GDKit, and if any of those extensions require some type of configiration the configuration can be specified through this dictionary manually in code, or by loading in a plist.
